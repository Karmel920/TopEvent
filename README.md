# TopEvent

## Usage:
* `git clone https://github.com/Karmel920/TopEvent.git`
* `cd TopEvent`
* `pip install virtualenv`
* `virtualenv envname`
* `envname\scripts\activate`
* `pip install -r requirements.txt`

to run:
* `python manage.py runserver`

## Technologies Used
- Django 4.1.7
- HTML5
- CSS3
- JS ES6
